# Claude Conversations Viewer

Browse, search, and analyze your exported Claude chat history in a local web UI. Optionally use an LLM to extract software project ideas from your conversations.

## What it does

- **Conversations viewer** — browse all your Claude chats organized by topic (AI, programming, finance, etc.), with full-text search and message-level display
- **Projects viewer** — extracts every software idea mentioned across your conversations and groups/deduplicates them using an LLM

## Setup

### 1. Export your Claude conversations

1. Go to [claude.ai](https://claude.ai) → Settings → Privacy → **Export data**
2. Download and unzip the archive
3. Copy `conversations.json` into this folder (`claude_text_data/`)

> `conversations.json` is listed in `.gitignore` — it will never be committed.

### 2. Install dependencies

The conversation processor only uses the Python standard library. The project idea extractor needs one extra package:

```bash
pip install openai python-dotenv
```

### 3. Process your conversations

Run this from the `claude_text_data/` directory:

```bash
python process_conversations.py
```

This reads `conversations.json` and writes:
- `conversations_viewer/index.json` — lightweight index of all conversations
- `conversations_viewer/topics/*.json` — full conversation data split by topic

### 4. Open the viewer

```bash
cd conversations_viewer
python serve.py
```

Then open **http://localhost:8765** in your browser.

---

## Optional: extract project ideas

This step uses the Gemini API (free tier works) to scan your conversations and pull out every software project idea you've discussed with Claude.

### Get a Gemini API key

1. Go to [aistudio.google.com](https://aistudio.google.com) → Get API key
2. Create a `.env` file in `claude_text_data/`:

```
GEMINI_API_KEY=your_key_here
```

### Run the extractor

```bash
python extract_project_ideas.py
```

This reads the topic files generated in step 3 and writes:
- `conversations_viewer/projects_all.json` — every idea found (including variants)
- `conversations_viewer/projects.json` — deduplicated and grouped

Progress is checkpointed, so you can safely Ctrl+C and resume. The extractor is rate-limit aware and processes conversations in batches.

### View your projects

Open **http://localhost:8765/projects.html** or click the **Projects** link in the conversations viewer.

## File structure

```
claude_text_data/
├── process_conversations.py     # Step 1: process export → viewer data
├── extract_project_ideas.py     # Step 2 (optional): extract project ideas
├── conversations.json           # Your Claude export — gitignored, add your own
├── projects.json                # Your Claude projects export — gitignored
├── conversations_viewer/
│   ├── serve.py                 # Local HTTP server
│   ├── index.html               # Conversations browser UI
│   ├── projects.html            # Project ideas UI
│   ├── index.json               # Generated by process_conversations.py
│   ├── topics/                  # Generated by process_conversations.py
│   ├── projects.json            # Generated by extract_project_ideas.py
│   └── projects_all.json        # Generated by extract_project_ideas.py
```

## Notes

- All processing is local — your conversations never leave your machine except for the optional Gemini API calls (which send truncated conversation text)
- The topic classifier is keyword-based and runs entirely offline
- Processing ~5,000 conversations takes about 30 seconds; idea extraction depends on API rate limits (estimate ~10 min per 1,000 conversations on the free tier)
